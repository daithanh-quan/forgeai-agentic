# Python API Implementation

Use this skill for Python backend API changes.

## Checklist

- Identify the framework and dependency manager from project files.
- Validate request models, serializers, and response schemas.
- Preserve public status codes and response shapes unless intentionally
  changed.
- Update tests for route and service behavior.
- Run existing formatter, linter, type checker, and test commands when
  available.
