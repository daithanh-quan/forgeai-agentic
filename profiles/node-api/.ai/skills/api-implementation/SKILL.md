# API Implementation

Use this skill for Node.js backend API changes.

## Checklist

- Find the route/controller boundary before editing services or data access.
- Validate request inputs at the boundary.
- Preserve public response shape unless the task explicitly changes it.
- Update specs, examples, or OpenAPI artifacts when contracts change.
- Add or update route/service tests for behavior changes.
- Check auth, permissions, rate limits, and error handling.
