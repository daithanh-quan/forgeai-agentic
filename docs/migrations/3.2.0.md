# Migration to ForgeAI 3.2.0

Version 3.2.0 adds generated dependency evidence and compiled-context schema 1,
including applicable rules and read-only diagnostic evidence.

## Upgrade

```bash
npx forgeai-agentic-init@latest --upgrade
forgeai-init --refresh-codegraph
forgeai-init --check-codegraph
```

The upgrade preserves curated CodeGraph content, project context, memory,
routing, security policy, and run state. `dependency-graph.json` is generated
project evidence and is also preserved until an explicit refresh.

## Behavior change

`--context-pack` no longer treats a manually populated `graph.json` as enough
dependency evidence. It fails with an actionable message when
`.ai/codegraph/dependency-graph.json` is missing, invalid, or stale.

No automatic source scan occurs during upgrade or context compilation.
